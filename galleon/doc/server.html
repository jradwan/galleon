Server

Name: The name of the server sent to TiVo.
Version: The JavaHMO version.
Reload: The default reload period for internet content to be downloaded.
TiVo Beacon: Whether TiVo Beacon service should be used.
Streaming Proxy: Whether all streaming stations should be proxied through the JavaHMO server.
Generate Thumbnails: Whether thumbnail versions of images are generated at system startup.
Port: The port on the local PC used to communicate with TiVo HMO.
IP Address: For computers with multiple network interface cards, it might be neccessary to specify which network interface is on the private network.
Netmask: The network mask of the chosen network interface IP Address. A netmask must be specified if an IP Address is specified.

Note: Any changes to the server properties requires JavaHMO to be restarted.
